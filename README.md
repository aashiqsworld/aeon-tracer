# Aeon Tracer

![ray traced scene](readme-assets/image3.png)

A C++ Ray Tracer.

![ray traced cube](readme-assets/renderCube.png)

Current features:
- ray traced meshes (triangular)
- ray traced spheres
- bitmap support
- materials (diffused, metal, dielectric)
- reflection/refraction
- physically-based camera
- ppm image output
